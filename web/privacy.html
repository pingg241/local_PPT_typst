PPTypst 隐私政策

最后更新：2026 年 1 月 26 日

PPTypst 是一个 PowerPoint 加载项，可以让你直接在演示文稿里使用 Typst 插入和编辑公式。它支持实时预览，也支持从文件生成内容。 本项目不隶属于 Typst 官方，而是社区驱动的实现。

这个项目是完全 开源 且基于 MIT 协议发布的。 我们不会存储任何用户数据或个人信息。你的 Typst 代码和生成结果都保留在本地机器以及 PowerPoint 文件中。 唯一可能产生的额外数据，是 GitHub Pages 在加载加载项界面时记录的标准访问日志，因为界面页面托管在 GitHub Pages 上。

概述

本隐私政策说明当你使用 PPTypst（以下简称“本应用”）时，我们如何处理相关信息。

信息收集与使用

我们不会收集、存储或传输任何个人信息或用户数据。

PPTypst 加载项的实际工作过程完全发生在你的本地机器上 （启动时加载一份包含 HTML/JS/CSS 的静态页面除外）。当你使用它向 PowerPoint 插入公式时：

  • 你编写的全部 Typst 代码都保留在本地设备上。
  • 所有生成出的图像和内容都留在 PowerPoint 文档内部。
  • 不会有数据发送到我们的服务器，因为我们没有后端服务器。
  • 应用内没有任何分析或跟踪机制。
  • 也不会创建或维护任何用户账号。

第三方服务

PPTypst 的界面托管在 GitHub Pages 上，属于静态网站。 当你在 PowerPoint 中加载该加载项时，GitHub Pages 可能会记录标准 Web 访问日志，其中可能包括：

  • 你的 IP 地址
  • 浏览器类型与版本（也就是 PowerPoint 内部 WebView 的信息）
  • 访问日期与时间

这些日志由 GitHub 收集和管理，并不是由我们保存。 关于 GitHub 如何处理这些数据，请参阅 GitHub 隐私声明。

数据安全

由于我们不收集或存储任何个人信息，因此不存在需要我们额外保护的用户数据。 你使用本应用创建的内容都保留在自己的 PowerPoint 文件中，也不存在需要你向我们申请访问、更正或删除的个人信息。

隐私政策变更

我们可能会不定期更新本隐私政策。任何变更都会发布在本页面，并同步更新“最后更新”日期。 建议你在需要时重新查看本页面。

联系我们

如果你对本隐私政策或 PPTypst 加载项有任何问题，可以通过 GitHub 仓库 联系项目维护者。
