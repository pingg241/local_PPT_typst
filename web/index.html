PPTypst | 把 Typst 带进 PowerPoint

PPTypst 是一个 PowerPoint 加载项，可以让你直接在演示文稿里使用 Typst 插入和编辑公式。它支持实时预览，也支持从文件生成内容。 这个项目并不隶属于 Typst 官方，而是社区驱动的实践，用来把 Typst 的排版能力带到 PowerPoint 中。

项目仓库位于 GitHub。 目前它还没有上架 PowerPoint 加载项市场，安装和本地测试方式请参考仓库说明。

PowerPoint 会连接到这个页面来显示加载项界面。所有数据都保留在你的本地机器上； 这里没有后端服务，页面只是托管在 GitHub Pages 上的静态站点。更多说明见 隐私政策。
